# Mediacore classes {#mediacore-classes}

You can use the Primetime Player API to customize the behavior of the player.

To see the complete API documentation for TVSDK, go to the [Adobe Primetime API References](https://help.adobe.com/en_US/primetime/api/index.html#api-Adobe_Primetime_API_References).

These classes describe your media player and its resources. 
Package: [com.adobe.mediacore](https://help.adobe.com/en_US/primetime/api/psdk/javadoc_1.4/com/adobe/mediacore/package-summary.html)
